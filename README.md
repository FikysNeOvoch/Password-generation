Пітч
Password Generation — це простий веб-сервіс, який дозволяє швидко створювати надійні випадкові паролі для захисту акаунтів. Продукт допомагає користувачам уникати слабких або повторюваних паролів і підвищує безпеку їхніх онлайн-даних.

Проблема
Багато користувачів використовують прості або однакові паролі для різних сайтів. Через це акаунти стають вразливими до злому, а особисті дані можуть бути викрадені.

Ціль першої ітерації
Перевірити, чи достатньо простого генератора паролів для швидкого створення безпечних паролів.

MVP
Мінімальна версія продукту включає:
генерацію випадкового пароля;
відображення згенерованого пароля на екрані;
можливість скопіювати пароль для використання.

Стейкхолдери
Основний стейкхолдер — користувач інтернет-сервісів, якому потрібен надійний пароль.
Другорядні — розробник проєкту та викладач, який оцінює результат роботи.

Критерії успіху
користувач може згенерувати пароль менш ніж за 10 секунд;
головний сценарій (генерація пароля) працює без помилок;
новий користувач розуміє інтерфейс без додаткових пояснень.
